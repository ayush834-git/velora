# Velora

Velora is a cinematic movie discovery project with a Next.js frontend and a minimal Express API scaffold.

## Frontend (apps/web)

```bash
npm install
npm run dev
```

Frontend runs on `http://localhost:3000`.

## Backend (apps/api)

```bash
npm run dev:api
```

API runs on `http://localhost:4000` by default and exposes:

- `GET /` -> `VELORA API running.`
